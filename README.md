# 🌐 HDC Host - Hospedagem de Sites

O HDC Host é um projeto de site institucional para uma provedora de serviços de hospedagem e domínios. O foco principal foi a criação de uma interface limpa, organizada e profissional.

## 🚀 Destaques do Projeto
* **Tabelas de Preços:** Estruturação de planos e serviços de forma clara para o usuário.
* **Layout Corporativo:** Uso de paleta de cores e tipografia voltadas para o setor de tecnologia.
* **Componentes de UI:** Seções de depoimentos, lista de diferenciais e rodapé completo.

## 🛠️ Tecnologias
* HTML5 (Semântico)
* CSS3 (Flexbox para o grid de serviços e planos)
* Google Fonts & Font Awesome (Iconografia)
